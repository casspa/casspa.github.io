Take Your Pick

E-Commerce Website



Project Brief

Take Your Pick’ is a fictional company that specialises in guitar accessories including guitar picks, pedals, cables and more. They are seeking a website that will allow them to expand their business online.

The scope was to create an e-commerce website that will display their products, allow users to add items to a shopping cart, accept orders from buyers and display the final order. The website will be a database driven website, implemented using PHP.


Technical Requirements

Language:

HTML/CSS – used to create the layout and styling of the website
PHP – handling the functionality of the website such as the store
JavaScript – used for client side validation for registering a new customer, etc.
SQL – used to develop the database


Security Requirements:
Hyper Text Transfer Protocol Secure (HTTPS) – to encrypt all client and server communication
Password Encryption – to protect user data
Login Authorisation – only those who have registered and entered the correct username and password will be able to login, incorrect details will be prompted to re-enter data and new customers will be prompted to register
