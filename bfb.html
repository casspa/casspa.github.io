Barefoot Bond

Unity Game



Project Brief

The idea for this project began between two siblings joking about making a Barefoot Skiing game featuring their lecturer. After enrolling in Web Development and Digital Media, we both needed a project that would cater to the requirements of both subjects, use some of our previously learnt skills and something we would be excited to create.

Our project is a 2D game which features the main character, Barefoot Bond, competing in a Barefoot Skiing competition. The game is being developed in Unity and hosted on a website that we will also be developing.
